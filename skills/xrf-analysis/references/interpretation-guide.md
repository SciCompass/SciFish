# Interpretation Guide

## Minimum reporting set

- visible sample metadata
- top reported oxide components with `mass%`
- low-level or near-threshold components called out conservatively
- scan label, scan range, point count, and relative intensity range
- explicit interpretation limits

## Reading rules

- Use the semi-quantitative table as the primary source for composition reporting.
- Use the scan trace only to describe the exported scan structure, not to identify mineral phases.
- If a reported component value is close to or below its detection limit, flag that uncertainty explicitly.
- Do not let the sample name override the measured result table.
- If a user asks for XRD-style conclusions, state clearly that XRF and XRD answer different questions.

## Writing discipline

- Facts first, interpretation second.
- Use `mass%` exactly as the export reports it.
- Say `reported as` or `suggests` when describing composition tendencies.
- State clearly when the export cannot confirm crystalline phase or mineral identity.
