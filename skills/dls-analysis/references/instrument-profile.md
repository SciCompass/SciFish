# Instrument Profile

## Scope

Dynamic light scattering (DLS) is commonly used to measure the hydrodynamic particle-size distribution of dispersed particles in liquids. When the instrument also includes electrophoretic light scattering capability, it may additionally report zeta potential and related mobility results.

## Confirmed in the current sample set

- The export contains a `DTS software` software trace.
- The real sample set currently confirms a particle-size intensity-distribution export.
- No direct zeta-potential numeric page or table has yet been confirmed in this archive.

## What the current data can support

- Main peak particle size
- Whether the distribution is relatively narrow or broad
- Whether a large-particle tail is present
- Whether the data suggests possible aggregates or a small coarse-particle fraction

## What the current data cannot support on its own

- True particle number distribution
- Exact concentration or mass fraction
- Surface-charge stability claims when zeta data is absent
- Chemical composition or crystal-phase identity
