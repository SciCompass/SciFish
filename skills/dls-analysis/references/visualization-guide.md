# Visualization Guide

## Figure type

- Main plot: particle-size intensity-distribution curve
- x-axis: `Particle size (nm)`
- y-axis: `Intensity distribution (%)`

## Axis rules

- Use a logarithmic x-axis because it better matches the observed size binning.
- Cover the full valid size range by default.
- If an unusually long tail is present, keep the full range but explain its impact in the prose.

## Annotation rules

- Mark the top `1` to `3` peaks.
- Label peak positions in `nm`.
- Avoid excessive text that obscures the curve.

## Output formats

- PDF: vector output
- PNG: 300 DPI preview
