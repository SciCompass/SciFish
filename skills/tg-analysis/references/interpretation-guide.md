# Interpretation Guide

## Minimum reporting set

- Test range: start and end temperature
- Initial and final mass percent
- Total mass loss percent
- Main stage count
- For each stage: onset, DTG peak temperature, end, mass-loss percent
- Final residue interpretation, if any

## Interpretation rules

- A small early loss below about 150 C is often consistent with moisture or residual solvent removal.
- A large mid-temperature loss with a strong DTG minimum is usually the main decomposition stage.
- Broad overlapping events should be described as overlapping, not as sharply separated stages.
- High final residue can be consistent with inorganic ash, fillers, or stable char, but the cause should not be overclaimed.

## Writing discipline

- Facts first, then interpretation.
- Use conservative verbs.
- Mention missing metadata such as atmosphere, heating rate, crucible type, or sample mass when absent.
- If curve quality is noisy, state that peak positions and boundaries are approximate.
