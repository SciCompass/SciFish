## Validation rules

- `emission_wavelength_nm` must be monotonic non-decreasing inside each scan.
- Median step size should be positive.
- Counts must be finite.
- Dominant peaks should be ranked by prominence.
- Comparison ratios are heuristic summaries only and do not imply absolute photophysical constants.

## Chinese report structure

The standardized Chinese Markdown report generated by `scripts/render_pl_report.py` should follow this section order:

1. `样品与数据概况`
2. `图谱结果`
3. `图谱图像`
4. `结构化分析`

The report should:

- embed the PNG figure inside the Markdown body
- keep scan-level facts and scan-to-scan comparison separate
- summarize only what is directly supported by the steady-state PL result
- avoid standalone “结论边界” sections in the final template
