# PL Feature Notes

## What this skill can safely report

- strongest emission wavelength
- top secondary local maxima
- broad versus sharply resolved emission behavior
- relative intensity differences between scans in the same archive

## What this skill must not overclaim

- exact emitting species
- trap-state origin
- aggregation state
- energy-transfer pathway
- lifetime component structure
- photoluminescence quantum yield

## Practical notes for the current dataset

- The scans are in the near-infrared region (`900-1600 nm`).
- Raw counts include negative values at some wavelengths, so weak shoulders near the baseline require caution.
- The current archive appears suitable for emission-maximum and relative-intensity reporting, not for transient fitting.
