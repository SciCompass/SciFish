# Measurement Modes

## TPR

- Typical use: reduction event screening under a reducing gas stream
- Safe reporting focus: reduction peak temperatures, relative event count, low- versus high-temperature reducibility trends
- Avoid: claiming exact oxidation state sequence or hydrogen consumption without calibration

## TPO

- Typical use: oxidation or coke burn-off event screening
- Safe reporting focus: oxidation peak temperatures and relative event structure
- Avoid: claiming exact carbon content or oxygen uptake from detector signal alone

## TPD

- Typical use: desorption profile screening for adsorbates or acid/base probe molecules
- Safe reporting focus: desorption temperature windows and relative event distribution
- Avoid: assigning exact desorbing species without corroborating metadata or known detector configuration

## Cross-mode rule

Do not reuse a TPR-specific interpretation template for TPD or TPO without confirming the file metadata and detector context first.
