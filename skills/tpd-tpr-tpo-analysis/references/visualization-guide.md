# Visualization Guide

## Figure specifications

- Default canvas: `6.8 x 4.6 inch`
- Font: `Arial`, `Helvetica`, or `DejaVu Sans`
- Main trace color: dark slate or deep blue
- Optional baseline or smoothed trace: muted gray
- Export both `.pdf` and `.png`

## Axis conventions

- X-axis: `Temperature (°C)`
- Y-axis: `Signal (a.u.)`
- Mark major derived event temperatures with thin vertical guide lines if needed

## Output files

- Primary: `output/figures/generic-tpd-tpr-tpo-analysis/{sample}-profile.pdf`
- Preview: `output/figures/generic-tpd-tpr-tpo-analysis/{sample}-profile.png`
