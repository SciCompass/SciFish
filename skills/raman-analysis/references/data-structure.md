## Validation rules

- `raman_shift_cm^-1` must be monotonic non-decreasing.
- Median step size should be positive.
- Intensity values should be finite.
- Detected bands should be ranked by descending prominence for reporting.
- `background_character` should distinguish resolved-band spectra from broad or fluorescence-dominated behavior.
- Carbon-screening flags are heuristic only and must never be treated as definitive identification.

## Chinese report structure

The standardized Chinese Markdown report generated by `scripts/render_raman_report.py` should follow this section order:

1. `样品与数据概况`
2. `图谱结果`
3. `图谱图像`
4. `结构化分析`

The report should:

- embed the PNG figure inside the Markdown body
- summarize dominant bands and broad-band/background behavior separately
- keep conclusions at the peak-screening level when no reference spectrum is provided
