# Band Tables

This file is intentionally conservative. Use it only for tentative screening and never as the sole basis for definitive assignment.

## General matching rules

- Prefer multi-band agreement over a single-band match.
- Compare the strongest observed bands first.
- Require both position agreement and a plausible relative-intensity pattern.
- If band offsets exceed a reasonable experimental tolerance, report the mismatch.

## Working tolerance

- Initial screening tolerance: about `+/- 20 cm^-1`
- Tighter interpretation requires expert confirmation and reference metadata
