Agent-native skills for

SciFish is a curated set of expert-validated Skills that turn raw materials-characterization instrument exports into journal-grade figures and structured analysis reports. Drop a Skill into any Skill-aware AI Agent and the Agent gains the analytical judgment of a domain specialist for that instrument.

Get started View on GitHub ↗

Works with Claude Code · Codex CLI

Self-contained & portable

Each Skill is a standalone directory: SKILL.md, scripts/, references/, assets/. Drop it into your Agent. No project glue code, no global config.

Conservative by construction

Every Skill enforces explicit separation between observed values and inferred assignments, and blocks common over-claims (phase identity from a single XRD scan; surface free energy from a sessile-drop screenshot).

Journal-grade figures

Every Skill emits a .pdf (vector, submission-ready) and a .png (300 DPI) figure pair, following its instrument-specific visualization conventions — axis orientation, palette, peak labels.

10 Skills in v1.0

Each iterated on real supplier exports and signed off by a domain expert. Click git or curl on any skill to copy the install command.

Wettability

contact-angle-surface-tension-analysis

Goniometer / drop-shape analysis. Extracts left/right contact angles from screenshot bundles, computes symmetry & wettability flags, and renders a fixed-section Chinese report.

Particle size

dls-analysis

Dynamic light scattering. Parses Malvern-style .xls exports, reports particle size distribution, PDI, and peak/tail screening with robustness checks.

Vibrational

ftir-analysis

FTIR spectra (.csv/.txt/ .xls(x)). Adaptive peak regression, broad-band flags, conservative functional-group screening.

Photoluminescence

pl-analysis

PL spectra. Emission band positions and intensities, publication-ready spectra, and a deliverable report.

Vibrational

raman-analysis

Raman spectra and .zip multi-spectrum bundles. Per-spectrum bands, cross-replicate consistency, and fluorescence-background warnings.

Thermal

tg-analysis

Thermogravimetry. Mass-loss steps and key temperatures with a structured report ready for materials-chemistry handoff.

Chemisorption

tpd-tpr-tpo-analysis

Temperature-programmed desorption / reduction / oxidation. Peak temperatures, integrated quantities, and engineered report output.

Optical

uv-vis-nir-drs-analysis

UV-Vis-NIR diffuse reflectance. Tauc fitting, Eg estimation, multi-sample overlay plots in publication style.

X-ray

xafs-analysis

X-ray absorption fine structure. E0 extraction, k-space SNR quality, R-space apparent peaks, with Athena/Larch-aligned interpretation language.

X-ray

xrf-analysis

X-ray fluorescence. Element list, relative content ranking, and conservative composition reading that refuses to misclaim mineral phase.

Install

Pick your Agent. Both install paths produce the same on-disk Skill.

Claude Code

Drop any Skill into ~/.claude/skills/:

git clone https://github.com/SciCompass/SciFish
cp -R SciFish/skills/raman-analysis ~/.claude/skills/

Or, one line:

curl -fsSL https://scicompass.github.io/SciFish/install.sh | bash -s raman-analysis

Restart Claude Code. The Skill is now invocable by name.

Codex CLI

Symlink the cloned skills/ directory into Codex's configured Skill path, or copy individual Skill directories there. Consult codex --help for the path on your version.

git clone https://github.com/SciCompass/SciFish
ln -s "$(pwd)/SciFish/skills/raman-analysis" ~/path/to/codex/skills/

Python dependencies

Skills' scripts assume a standard scientific Python stack. Install once:

pip install numpy pandas scipy matplotlib seaborn lmfit pymatgen \
  fabio h5py xraydb periodictable openpyxl chardet requests

Recommended Python: 3.8+. Some Skills (e.g. xafs-analysis) work best with Larch available. Each Skill's references/instrument-profile.md calls out optional deps.

Where outputs land

Every Skill writes outputs under output/ in your current working directory:

<your-cwd>/
└── output/
    ├── data/        # parsed/intermediate JSON
    ├── figures/     # journal-grade .pdf + .png pairs
    └── reports/     # final Markdown reports

Roadmap

  • More Skills. Bring the remaining 11 instruments (BET, NMR, XPS, XRD, HRMS, EPR, ICP-OES/MS, GPC, DSC, TG-DSC, EA) out of internal validation and into this repo.
  • Examples. Ship redacted example datasets and expected outputs alongside each Skill so users can verify their setup end-to-end.
  • CI smoke tests. Lint and dry-run every Skill on every PR.
  • Multilingual reports. English templates for the Skills that currently default to Chinese.

Contributing

Lowest-friction contributions

  • Peak / band / binding-energy / characteristic-temperature tables for an existing Skill
  • Edge-format support (vendor exports the parser fails on — attach a small redacted reproducer)
  • Bug fixes in scripts/ that preserve existing test cases

Bigger contributions

  • An entirely new instrument Skill — please open an Issue describing the instrument and your domain background before sending a large PR.
  • A new visualization-guide.md for an existing Skill if you publish in that field.
